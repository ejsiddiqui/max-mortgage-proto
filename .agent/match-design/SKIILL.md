# Match Design Skill

1. Read the reference screenshot or wireframe the user provides.
2. Match this reference design exactly. Before writing any code.
3. Identify ALL visual elements: spacing, icon sizes, column alignment, text widths, colors.
4. Implement changes across all relevant files.
5. Self-review: compare your output against the reference point-by-point.
6. List any deviations before presenting to user.
